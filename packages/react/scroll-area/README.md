# `react-scroll-area`

## Installation

```sh
$ yarn add @interop-ui/react-scroll-area
# or
$ npm install @interop-ui/react-scroll-area
```

## Usage

```js
import * as React from 'react';
import { ScrollArea, styles } from '@interop-ui/react-scroll-area';

function MyComponent(props) {
  return (
    //
  );
}
```
