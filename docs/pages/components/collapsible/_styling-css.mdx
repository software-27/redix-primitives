Each `Collapsible` component's DOM node has a data attribute that you can target to style via CSS.

```css
/* outer container styles */
[data-radix-collapsible] {
  margin-bottom: 1em;
}

/* button styles */
[data-radix-collapsible-button] {
  color: var(--color-button-text);
  font-weight: 700;
}

/* content  styles */
[data-radix-collapsible-content] {
  padding-top: 1em;
}
```

Alternatively, you can pass class names to each component.

```css
.collapsible {
  margin-bottom: 1em;
}

.collapsible__button {
  color: var(--color-button-text);
  font-weight: 700;
}

.collapsible__content {
  padding-top: 1em;
}
```

```jsx
<Collapsible className="collapsible">
  <CollapsibleButton className="collapsible__button">Greeting</CollapsibleButton>
  <CollapsibleContent className="collapsible__content">Hello there!</CollapsibleContent>
</Collapsible>
```
